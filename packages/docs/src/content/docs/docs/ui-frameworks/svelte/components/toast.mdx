#### Setup

Make sure to include the `Toaster` component inside the provider to enable toast
notifications.

For installation and setup instructions, see the
[Getting Started guide](/docs/ui-frameworks/svelte/getting-started).

> Recommended: Place this in `src/routes/+layout.svelte` (SvelteKit root layout)

```svelte
<script lang="ts">
	import { Provider, Toaster } from "@dxdns-kit/svelte"
</script>

<Provider>
	<Toaster />
	{@render children()}
</Provider>
```
