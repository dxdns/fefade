### Theme configuration

This setup allows the user to define **partial** theme colors. That means it's
not required to specify every color â€” only the ones you wish to customize. Any
colors not explicitly defined will automatically fall back to the default
values.

Import necessary modules and create a custom theme configuration.

```ts
import { createApp, h } from "vue"
import App from "./App.vue"
import { createTheme, FeflowProvider } from "./lib"

const app = createApp({
	render() {
		return h(
			FeflowProvider,
			{ theme: createTheme({ colors: { light: { bg: "orange" } } }) },
			{
				default: () => h(App)
			}
		)
	}
})

app.mount("#app")
```
