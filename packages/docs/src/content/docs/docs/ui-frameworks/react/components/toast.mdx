#### Setup

Make sure to include the `Toaster` component inside the provider to enable toast
notifications.

For installation and setup instructions, see the
[Getting Started guide](/docs/ui-frameworks/react/getting-started).

> Recommended: Place this in `src/main.tsx` (React root element)

```tsx
import { StrictMode } from "react"
import { createRoot } from "react-dom/client"
import App from "./App"
import { Provider, Toaster } from "@fefade/react"

createRoot(document.getElementById("root")!).render(
	<StrictMode>
		<Provider>
			<Toaster>
				<App />
			</Toaster>
		</Provider>
	</StrictMode>
)
```
