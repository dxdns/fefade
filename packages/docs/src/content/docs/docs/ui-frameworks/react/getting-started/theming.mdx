### Theme configuration

This setup allows the user to define **partial** theme colors. That means it's not required to specify every color â€” only the ones you wish to customize. Any colors not explicitly defined will automatically fall back to the default values.

Import necessary modules and create a custom theme configuration.

```tsx
import { StrictMode } from "react"
import { createRoot } from "react-dom/client"
import App from "./App"
import { FeflowProvider, createTheme } from "@dxdns/feflow-react"

const theme = createTheme({
	colors: { light: { bg: "orange" } }
})

createRoot(document.getElementById("root")!).render(
	<StrictMode>
		<FeflowProvider theme={theme}>
			<App />
		</FeflowProvider>
	</StrictMode>
)
```
