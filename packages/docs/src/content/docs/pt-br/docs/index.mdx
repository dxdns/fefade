**fefade** é uma biblioteca de componentes de interface de usuário leve e livre
de dependências, focada em desempenho e simplicidade.

> Mínima, acessível, livre de dependências — criada por um desenvolvedor que ama
> interfaces de usuário limpas.

### Por que o fefade?

- Design leve
- Estilos isolados usando Módulos CSS
- Simples de ler, simples de usar

### Recursos

- **Sem dependências externas**
- **Estilos com escopo definido com Módulos CSS** (sem conflitos globais)
- Integração opcional com o Tailwind (não necessária)
- **Fácil personalização de temas** — desenvolvido para oferecer suporte a temas
  prontos para uso
- **Temas escuros e claros incluídos** por padrão, alternáveis ​​por meio de
  propriedades simples ou variáveis ​​CSS
- Totalmente tipado com TypeScript
- Slots de componentes e propriedades personalizáveis
- Desenvolvido para a experiência do desenvolvedor (DX)
